# addIcon

```ts
export function addIcon(iconId: string, svgContent: string): void;
```

Adds an icon to the library

## Parameters

| Parameter | Description |
|-----------|-------------|
| `iconId` | the icon ID |
| `svgContent` | the content of the SVG, without the &lt;svg&gt;. Must fit viewBox="0 0 100 100". |
